
# Math

## .ceil

返回大于或等于一个给定数字的最小整数

```js
console.log(Math.ceil(7.004));
// expected output: 8
```

## .floor

返回小于或等于一个给定数字的最小整数

```js
console.log(Math.ceil(7.004));
// expected output: 7
```

## .round

Math.round(x) 函数返回一个数字四舍五入后最接近的整数。

x: 一个数值

```js
Math.round(20.49) // 20
Math.round(20.5) // 21
Math.round(-20.5) // -20
Math.round(-20.51) // -21
```
